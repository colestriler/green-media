def eliminate_excess(series):
    """In news headlines, there are many excess punctuation.
    This function removes all of that."""
    punct_re = r'[^A-Za-z\d\s]'
    #series = series.apply(lambda x: x.replace("", ""))
    series = series.str.replace(punct_re, " ")
    return series.str.lower()

def count_frame(df, n):
    """Takes in a data frame and int n:
    
    DataFrame/ df: dataframe with HEADLINE column
    int/ n: number of words to choose
    r/type: 2 column dataframe most common words & counts"""
    
    counts = to_dict(df.headline)
    frame = pd.DataFrame.from_dict(counts, orient='index')
    frame = frame.sort_values(0, ascending=False).reset_index().head(n)
    frame = frame.rename(columns={'index': 'word', 0:'count'})
    return frame

def to_dict(series):
    """Takes in a pd.Series of Strings and
        1) Takes every string
        2) Splits into individual words
        3) Checks if word is a stopword
        4) if not a stopword, add it to a dictionary
        5) If it is already in the dictionary, add a count"""
    dic = {}
    array = series.values
    for num in range(len(array)):
        headline = array[num].split()
        for word in headline:
            if word not in stopwords:
                if word in dic:
                    dic[word] += 1
                else:
                    dic[word] = 0
    return dic

News Polarity¶

It turns out that we can use the words in news headlines to calculate a measure of the sentiment of the headline. For example, the sentence "I love America!" has positive sentiment, whereas the sentence "I hate taxes!" has a negative sentiment. In addition, some words have stronger positive / negative sentiment than others: "I love America." is more positive than "I like America."

We will use the VADER (Valence Aware Dictionary and sEntiment Reasoner) lexicon to analyze the sentiment of the news headlines. VADER is a lexicon and rule-based sentiment analysis tool that is specifically attuned to sentiments expressed in social & news media which is great for our usage.
